# データベース SaaS どれにしようか問題

## 1. SurrealDB Cloud

### 利点

- スキーマで入れ子のUnion型をサポートしている
- 環境依存が少なめ

### 欠点

- 資料が少ない

## 2. Xata

### 利点

- 素のPostgreSQLをそのまま使える
- データを含めてブランチを作ることができる
- 環境依存が少ない

### 欠点

- 資料が少ない

## 3. Supabase

### 利点

- よく使われている
- 東京リージョンがある
- 管理画面のUIが使いやすい
- 環境依存が少ない

### 4. Deno KV

### 利点

- Deno から扱いやすい
- スケールしやすい
- シンプルな仕組みなため, 他への移植が簡単

### 欠点

- 自分で検索用のインデックスを保存する処理をする必要がある
- いろんなパターンのクエリをうまく高速化するのは難しい
- ベータ版で公式もやる気があるのかどうか分からない

# 結論

少し冒険をして SurrealDB Cloud にしようと思ったが,
無料枠のDBの接続キーをなぜか失ってしまったので Deno Deploy の Deno KV を利用する
