# データベース SaaS どれにしようか問題

## 1. SurrealDB Cloud

### 利点

- スキーマで入れ子のUnion型をサポートしている
- 環境依存が少なめ

### 欠点

- 資料が少ない

## 2. Xata

### 利点

- 素のPostgreSQLをそのまま使える
- データを含めてブランチを作ることができる
- 環境依存が少ない

### 欠点

- 資料が少ない

## 3. Supabase

### 利点

- よく使われている
- 東京リージョンがある
- 管理画面のUIが使いやすい
- 環境依存が少ない

# 結論

少し冒険をして SurrealDB Cloud にする
